## ⚡ Architectural & Interactive Features

### 1. Retro Draggable Terminal Shell
A mock terminal console that mimics a modern Linux system environment. It is fully draggable on desktop/mobile screens:
* **Interactive Command Line**: Type active commands in the prompt (`help`, `about`, `skills`, `projects`, `experience`, `certifications`, `lofi`, `clear`) and hit enter to print colorized, custom-formatted telemetry.
* **Window Controls**: Click the red control dot to hide, the yellow dot to minimize into a compact header, or the green dot to maximize into a high-utility fullscreen panel.

### 2. 3D Flippable Visa Credit Card Widget
A tactile developer credit card that serves as an interactive contact gateway:
* **Front View**: Features a realistic gold chip, custom brackets `[JAVA]`, and core email details.
* **Back View**: Tapping/hovering triggers a smooth 3D Y-axis rotation revealing an authorized signature strip, CVV, and an integrated quick email message form.
* **Feedback Loops**: Submitting a message fires custom loaders and logs automated signal responses into the terminal console history!

### 3. Ambient Lofi Audio Player
* Plays/pauses a soft synthwave lofi stream directly via the navigation bar.
* Features a responsive CSS-animated equalizer visualizer that bounces in sync with stream playback status.

## 🛠️ Installation & Local Run

To preview the portfolio locally on your machine, simply serve the folder:
1. Double-click `index.html` to open it in your browser.
2. Alternatively, run a simple Python local server:
   ```bash
   python -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser.
